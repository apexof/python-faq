#### Что такое Python?
**Python** — высокоуровневый интерпретируемый язык с динамической строгой типизацией.
#### Python компилируемый или интерпретируемый язык?
Питон — гибрид. Когда ты запускаешь `.py`, происходит два шага:

**1. Компиляция в байткод.** Питон переводит код в **байткод** — промежуточные инструкции. Это не машинный код для процессора, а команды для «виртуальной машины Питона». Байткод кэшируется в папке `__pycache__` (файлы `.pyc`), чтобы при следующем запуске не компилировать заново.
**2. Интерпретация.** Виртуальная машина Питона читает этот байткод инструкция за инструкцией и исполняет.

Считается интерпретируемым потому что нет отдельного шага сборки — запустил файл, и он работает. Компиляция в байткод спрятана и происходит сама.
#### Почему Python медленнее других языков?
- **Интерпретация байткода** — процессор исполняет цикл интерпретатора, а не код напрямую (в C/Go — сразу машинный код).
- **Динамическая типизация** — типы проверяются в рантайме на каждой операции (в C тип известен при компиляции).
- **Всё — объект** — даже `int` это объект с заголовком и счётчиком ссылок.
- **GIL** — нет настоящего параллелизма потоков на CPU-задачах.
- **Управление памятью** — постоянное обновление счётчика ссылок + работа сборщика мусора.
- **Нет JIT по умолчанию** (до CPython 3.13) — горячий код не компилируется в машинный.
