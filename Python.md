**Python** — высокоуровневый интерпретируемый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью.

## Особенности языка

| Особенность                       | Что это значит                                                                                                                                                                     |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Интерпретируемый                  | Код выполняется построчно интерпретатором ([[Реализации интерпретатора Python\|CPython]] компилирует исходник в байт-код, который исполняет виртуальная машина — без отдельного шага компиляции в машинный код, как в C) |
| Динамическая типизация            | Тип переменной определяется во время выполнения, а не объявляется заранее: `x = 5; x = "str"` — допустимо                                                                          |
| Строгая типизация                 | Неявные преобразования между несовместимыми типами не допускаются: `"1" + 1` вызовет `TypeError`                                                                                   |
| Автоматическое управление памятью | Объекты удаляются сборщиком мусора — счётчик ссылок (reference counting) + generational garbage collector для циклических ссылок                                                   |
| [[GIL]] (Global Interpreter Lock) | В [[Реализации интерпретатора Python\|CPython]] только один поток может выполнять байт-код Python в любой момент времени — влияет на многопоточность при CPU-bound задачах          |
| Мультипарадигменность             | Поддерживает объектно-ориентированный, функциональный и процедурный стили в одном языке                                                                                            |
| "Batteries included"              | Богатая стандартная библиотека (`os`, `itertools`, `collections`, `asyncio` и др.) без необходимости ставить сторонние пакеты для базовых задач                                    |
| Всё — объект                      | Числа, функции, классы, модули — все являются объектами с типом, атрибутами и идентификатором (`id()`)                                                                             |

```python
# ✅ динамическая типизация — переменная меняет тип
x = 5
x = "now a string"

# ❌ строгая типизация — неявного преобразования нет
"1" + 1  # TypeError: can only concatenate str (not "int") to str
```

## Зачем используется

- **Backend / web** — Django, Flask, FastAPI.
- **Data science и анализ данных** — pandas, NumPy, Jupyter.
- **Machine learning / AI** — PyTorch, TensorFlow, scikit-learn.
- **Автоматизация и скрипты** — системное администрирование, обработка файлов, парсинг.
- **Тестирование** — как язык для написания тестов и тест-фреймворков (pytest).

## Python 2 vs Python 3

Python 2 официально снят с поддержки 1 января 2020 года. Python 3 не обратно совместим с Python 2 (например, `print` — функция, а не оператор; `/` — всегда true division). Актуальная разработка ведётся только в линейке Python 3.x.
