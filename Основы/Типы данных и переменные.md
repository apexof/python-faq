**Переменная** — это имя, связанное с объектом в памяти. В Python переменная не имеет собственного типа — тип принадлежит объекту, на который она ссылается.

## Объявление переменных

```python
x = 10          # ✅ тип определяется автоматически (динамическая типизация)
name = "Alex"
is_active = True

x: int = 10     # ✅ аннотация типа — необязательна, не проверяется во время выполнения
```

## Базовые типы данных

| Тип | Пример | Описание |
|---|---|---|
| `int` | `42`, `-7` | целое число, неограниченный размер |
| `float` | `3.14`, `1e10` | число с плавающей точкой |
| `complex` | `3+4j` | комплексное число |
| `bool` | `True`, `False` | логический тип, подтип `int` |
| `str` | `"hello"` | строка, неизменяемая |
| `NoneType` | `None` | отсутствие значения |

```python
print(type(True))      # <class 'bool'>
print(isinstance(True, int))  # True — bool наследуется от int
```

## Изменяемые и неизменяемые типы

| Изменяемые (mutable) | Неизменяемые (immutable) |
|---|---|
| `list`, `dict`, `set`, `bytearray` | `int`, `float`, `str`, `bool`, `tuple`, `frozenset`, `bytes` |

```python
a = [1, 2, 3]
b = a
b.append(4)
print(a)  # [1, 2, 3, 4] — ✅ список изменяемый, a и b ссылаются на один объект

s = "hello"
s += " world"  # ❌ не изменение строки, а создание новой
```

## Динамическая типизация

Переменная может ссылаться на объекты разных типов в течение жизни программы.

```python
x = 10
x = "now a string"  # ✅ допустимо — типизация динамическая
```

## Идентификатор и сравнение объектов

- `id(obj)` — адрес объекта в памяти.
- `is` — сравнивает идентичность объектов (тот же ли это объект).
- `==` — сравнивает значения.

```python
a = [1, 2]
b = [1, 2]
print(a == b)  # True — равны по значению
print(a is b)  # False — разные объекты
```

## Приведение типов

```python
int("42")     # 42
str(42)       # "42"
float("3.14") # 3.14
bool(0)       # False
bool("")      # False
bool([])      # False
```

⚠️ `bool()` возвращает `False` для всех "пустых" значений: `0`, `0.0`, `""`, `[]`, `{}`, `set()`, `None`.

## Проверка типа

```python
type(x)            # точный тип объекта
isinstance(x, int)  # ✅ проверка с учётом наследования — предпочтительнее type()
```
