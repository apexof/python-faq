Задача 7
# Что выведет код?

print(0.25 + 0.25 == 0.5)
# True
print(0.1 + 0.2 == 0.3)
# False
​
Почему второе False? Потому что float числа хранятся с погрешностью из-за особенностей вычислений с плавающей точкой. НО если число является степенью 2, как в примере 0.25 == 2**(-2), то операции с такими числами - точные.
 Лучше использовать decimal.Decimal для точных вычислений.
