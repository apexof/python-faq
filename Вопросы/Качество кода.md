#### Type hints (аннотаций типов)
Type hints — это синтаксис для указания ожидаемых типов переменных, аргументов и возвращаемых значений. Они не влияют на выполнение, но используются статическими анализаторами (mypy, pyright) и IDE.

```python

from typing import List, Dict, Optional, Union  # 3.5+ типы надо импортировать

def process_data(items: List[str], limit: int = 10) -> Dict[str, int]: 
	return {item: len(item) for item in items[:limit]}

```

```python
# 3.10+ типы с маленькой буквы
def process(items: list[str], flag: bool) -> dict[str, int]:
    return {}

def find(user_id: int) -> str | None:  # вместо Optional
    return None
```

Type hints дают:
Автодополнение в IDE
Раннее обнаружение ошибок
Самодокументирование кода
Рефакторинг с уверенностью

#### mypy
Статический анализатор типов.

Если у нас есть аннотация типов, то мы можем запустить mypy и он найдет места, где мы передали неверный тип объекта

#### Pydantic
Валидация данных во время выполнения + автоматическое преобразование типов:

Pydantic = надежность данных. Вместо ручных проверок и краха в продакшене, вы получаете:

✅ Автоматическую валидацию
✅ Преобразование типов
✅ Понятные ошибки
✅ IDE подсказки
✅ Автодокументацию (OpenAPI в FastAPI)

Критически важен для работы с внешними данными: API, БД, конфиги, файлы.

#### Что такое линтеры? 
Линтер (linter) - инструмент статического анализа кода, который:

Находит ошибки до выполнения программы
Проверяет стиль кода
Выявляет потенциальные баги
Обеспечивает единообразие кода в команде

#### Flake8 
Объединяет PyFlakes (логические ошибки) + pycodestyle (PEP 8) + McCabe (сложность)

#### Ruff
**Ruff** — это линтер и форматтер для Python, написанный на Rust. Главная фишка — скорость: он в десятки-сотни раз быстрее аналогов на Python.

**Что заменяет:**  
Ruff объединяет в одном инструменте функциональность множества раздельных тулов — Flake8 (+ десятки его плагинов), isort (сортировка импортов), pydocstyle, pyupgrade, autoflake, и частично Black (форматирование).

#### isort

проверка сортировки импортов

isort автоматически сортирует и группирует импорты по стандарту PEP 8:
Стандартная библиотека
Сторонние библиотеки
Локальные модули
