Что делает функция enumerate?
`enumerate` — это встроенный класс, который принимает итерируемый объект и возвращает новый итератор, добавляя к каждому элементу целочисленный счётчик: на каждом шаге `next()` он отдаёт кортеж `(счётчик, элемент)` и увеличивает счётчик на 1 на каждой итерации.

Счётчик по умолчанию начинается с 0, но можно передать вторым аргументом `start`. Например, `enumerate(iterable, start=1)` начнёт счёт с 1. `start` влияет только на значение счётчика, а не на то, с какого элемента начинается перебор.
Если нужно перебирать не сначала нужно также взять стайс от итерируемого объекта через сред [:2] или itertools.islice (работает лениво)

Что делает функция zip ?
`zip` — это встроенный класс, который принимает произвольное число итерируемых объектов и возвращает новый итератор: на каждом шаге `next()` он берёт по одному следующему элементу из каждого переданного объекта и отдаёт их кортежем.
```python
list(zip([1, 2, 3], ['a', 'b', 'c']))
# [(1, 'a'), (2, 'b'), (3, 'c')]
```
Если объекты разной длины, итерация обрывается по самому короткому (лишнее отбрасывается), а при `strict=True` (Python 3.10+) — возбуждается `ValueError`.
