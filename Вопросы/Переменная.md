#### Что такое переменная?
Переменная - это ссылка на объект в памяти
#### Что происходит когда мы присваиваем одну переменную в другую?
Когда вы копируете переменную, вы копируете ссылку, а не сам объект.
#### Глобальные и локальные переменные
Локальная переменная — создаётся внутри функции и видна только там.
Глобальная переменная — создаётся вне функции и доступна везде.
Чтобы изменить глобальную переменную в функции, надо писать `global x`.
Чтобы изменить локальную переменную в объемлющей функции, надо писать `nonlocal x`

#### Почему не рекомендуется изменяемый тип как значение по умолчанию?
Данные изменяются и эти изменения сохраняются между вызовами. Если попытаться изменить неизменяемый объект то будет создан новый объект и локальная переменная будет указывать не него. 

```python
# Неправильно: 
def add_item(item, target_list=[]): # Опасно! 
	target_list.append(item) 
	return target_list 

# Правильно: 
def add_item(item, target_list=None): 
	if target_list is None: 
		target_list = [] 
	target_list.append(item) 
	return target_list
```
